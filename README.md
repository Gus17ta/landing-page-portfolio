# 🌐 Landing Page - Gustavo Brito

Landing page profissional desenvolvida com React e CSS3.

## 🚀 Demo
[Ver projeto ao vivo](https://landing-page-portfolio-red.vercel.app)

## 🛠️ Tecnologias
- React
- CSS3
- Vite

## ✨ Funcionalidades
- Design moderno e responsivo
- Seção hero com call-to-action
- Cards de serviços
- Layout adaptado para mobile e desktop

## 📦 Como rodar localmente
```bash
git clone https://github.com/Gus17ta/landing-page-portfolio
cd landing-page-portfolio
npm install
npm run dev
```
